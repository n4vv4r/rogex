# ROGEX.NET
(Hecho en mi episodio maníaco de bipolaridad tipo 1)

## DESCRIPCION
Rogex es una plataforma web desarrollada en PHP, JavaScript y MySQL orientada a interacción social, publicación de contenido y generación de quizzes, con sistema de usuarios, perfiles, feed, respuestas, subida de archivos y módulos experimentales. Este repositorio contiene la base del proyecto y su lógica principal; se excluyen deliberadamente directorios como space/, uploads/ y archivos de configuración sensibles (credenciales, claves API y datos de entorno) por razones de tamaño, privacidad y seguridad.
